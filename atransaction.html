    What are transfers with an execution date?

    Transfers with an execution date are transfers where you enter an exact date of execution: this is the date on which the money leaves your account.

    How do I create a transfer with an execution date?

    To make a transfer with an execution date, you indicate the date on which this transfer should be executed.

    Your transfer is automatically submitted for execution at the bank on the given execution date. It will then appear in the history of the account concerned under Transactions still to be processed.

    How do I monitor this?

    • Your transfer will be offered for execution automatically on the designated execution date.
    • On the execution date itself, your transfer will appear in the history for the account concerned under Transactions still to be processed.
    • However, if a transfer is rejected during this technical check, it usually appears after a few minutes in the Envelope in red. The reasons for rejection and further instructions are shown in the Details screen for the transfer.

    How do I view a transfer with an execution date?

    You can only view transfers with an execution date in PC banking after they have been entered and before they have been executed. Transfers with an execution date are displayed in PC banking immediately after entry:

    1. Select Accounts and cards in the left-hand menu.
    2. In the My PC banking menu, select the option Transfers with an execution date.
      You will see an overview of the transfers with an execution date that will be executed with the selected account. Transfers that are due to be executed on the same day will not appear in this list. You can select a different account in the selection list.
    3. Select Details in the Tools menu next to the transfer for which you would like to view the details. The Details screen will appear.

    How do I change a transfer with an execution date?

    You can only view transfers with an execution date in PC banking after they have been entered and before they have been executed. Transfers with an execution date are displayed in PC banking immediately after entry:

    1. Select Accounts and cards in the left-hand menu.
    2. In the My PC banking menu, select the option Transfers with an execution date. You will see an overview of the transfers with an execution date that will be executed with the selected account. Transfers that are due to be executed on the same day will not appear in this list. You can select a different account in the selection list.
    3. Select Details in the Tools next to the transfer for which you would like to view the details. The Details screen will appear.

    How do I delete a transfer with an execution date?

    You can only delete transfers with an execution date in PC banking after they have been entered and before they have been executed.

    1. Select Accounts and cards in the left-hand menu.
    2. In the My PC banking menu, select the option Transfers with an execution date.
      You will see an overview of the transfers with an execution date that will be executed with the selected account. Transfers that are due to be executed on the same day will not appear in this list. You can select a different account in the selection list.
    3. Select Delete in the Tools menu next to the transfer you would like to delete.
      The Delete transfer with an execution date screen will appear.
    4. Check the details of the transfer, enter an electronic signature and click on Confirm. The transfer with an execution date will be deleted.

    What can I put in a free message?

    The maximum length is 140 characters. Only the following characters are accepted:

    a-z A-Z 0-9 / - ?: () . , ' +

    If you would like to specify an amount, use 'euro' or 'EUR' to replace the '�' character.

    What is an electronic signature?

    An electronic signature is a code of up to 8 digits that you receive for each transaction you sign using your card reader and BNP Paribas Fortis bank card.

    • Follow the step-by-step on-screen instructions until this code appears on the screen of your card reader.
    • Finally, enter this code in Electronic signature and click on Confirm.

    To which accounts can I execute a transfer?

    • Transfer from your current account
      • to an account of a third-party beneficiary
        • which is already known and can be found in the list of known beneficiaries
        • which is not yet known and which you must add to the list subject to an M2 signature before the transfer can be executed
      • to one of your other accounts
      • to a savings account
    • Transfer from your savings account
      • to a own current account
